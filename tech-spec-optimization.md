# 勇者斗恶龙 JRPG 优化 Tech Spec

## 问题诊断

### 🔴 严重问题
1. **代码结构混乱**：所有逻辑堆在一个 3128 行的 game.js 里，全局函数满天飞，没有模块化
2. **多个冗余文件**：game-full.js、game-pixel.js、game.js 功能重叠，不清楚哪个是主文件
3. **HTML 内嵌巨量 CSS+JS**：index.html 1833 行，CSS 和结构混在一起
4. **备份文件残留**：index.html.bak、game.js.backup 不应该在仓库里
5. **SVG 资源冗余**：很多角色有 v1 和 v2 两套 SVG，未清理

### 🟡 体验问题
1. **强制刷新缓存**：每次加载都在 URL 后加 `?nocache=timestamp`，用户体验差
2. **alert() 弹窗**：怪物详情用 alert 显示，太原始
3. **对话框内联样式**：剧情系统的 HTML 全用内联 style，不可维护
4. **没有错误处理**：资源加载失败没有降级方案
5. **移动端体验**：虽然声称移动端适配但缺少手势操作

### 🟢 可优化项
1. **性能**：没有资源懒加载，所有 SVG 一次性加载
2. **存档**：只有 localStorage，没有导出/导入功能
3. **可访问性**：没有键盘导航支持
4. **SEO/分享**：没有 OG meta、没有分享功能

## 优化方案

### Phase 1：代码清理重构（最高优先级）
- [ ] 删除冗余文件（game-full.js、game-pixel.js、*.bak、*.backup）
- [ ] 清理冗余 SVG 资源（保留 v2 版本）
- [ ] 把 index.html 的 CSS 抽成独立 styles.css
- [ ] 移除强制缓存刷新的 hack
- [ ] 把内联 style 的对话框改成 CSS class

### Phase 2：UI/UX 优化
- [ ] alert() 替换为游戏内模态框
- [ ] 改进怪物图鉴的详情展示（全屏面板，有动画）
- [ ] 改进剧情对话系统（更好的视觉效果）
- [ ] 添加设置面板（音量、画质、文字速度）
- [ ] 底部导航栏优化（加图标、动画反馈）

### Phase 3：新功能
- [ ] 存档导出/导入功能
- [ ] 成就系统界面
- [ ] 键盘快捷键支持
- [ ] 战斗速度调节（1x/2x/3x）

## 验收标准
1. 项目只保留必要文件，无冗余
2. CSS 独立文件，index.html < 200 行
3. 所有 alert() 被游戏内 UI 替代
4. 移动端体验流畅
5. 游戏功能完整无回退
