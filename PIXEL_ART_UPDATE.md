# 勇者斗恶龙 JRPG - 像素画风格更新说明

## 更新时间
2026-02-25

## 主要改动

### 1. 像素画风格升级
- **字体**: 使用 Google Fonts 的 "Press Start 2P" 像素字体
- **角色精灵**: 所有角色使用SVG像素画，显示背面朝向敌人
- **怪物精灵**: 使用SVG像素画，显示正面朝向玩家
  - 绿史莱姆、蓝史莱姆、红史莱姆、黄金史莱姆、金属史莱姆
  - 幼龙/火龙/冰龙/雷龙
  - 骷髅兵、野狼
  - BOSS: 远古树精、洞穴巨魔、法老王、火焰领主、混沌魔王
- **8-bit调色板**: 限制颜色使用，模仿经典DQ风格

### 2. 战斗界面改进 (DQ经典布局)
- **勇者队伍**: 显示在左侧，使用背面精灵朝向敌人
- **敌人**: 显示在右侧，使用正面精灵朝向玩家
- **命令菜单**: 经典的4格命令菜单 (战斗/技能/物品/逃跑)
- **HP/MP条**: 复古风格像素条，带边框和高光效果

### 3. UI风格统一
- **像素字体**: 全站使用 Press Start 2P 字体
- **蓝色边框面板**: 所有面板使用经典DQ蓝色边框
- **像素风格按钮**: 3D效果的像素按钮，按下时有反馈
- **CRT扫描线**: 可选的复古CRT显示器扫描线效果

### 4. 动画效果
- **闪烁效果**: 攻击时敌人闪烁 (.flash-effect)
- **屏幕震动**: 受击时屏幕震动 (.shake-effect)
- **攻击闪光**: 攻击时的白光效果 (.attack-flash)
- **魔法光效**: 施法时的蓝色光晕 (.magic-effect)
- **受伤红色闪烁**: 角色受伤时红色闪烁 (.damage-effect)

### 5. 技术实现
- **SVG像素画**: 所有精灵使用SVG生成，支持无损缩放
- **移动端优化**: 响应式设计，确保手机显示正常
- **向后兼容**: 保留原有游戏逻辑和功能

## 文件结构
```
dragon-quest-jrpg/

```
