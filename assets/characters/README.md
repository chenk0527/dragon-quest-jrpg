# Character Sprites

Using emoji as fallback assets until pixel art is downloaded.

To add your own sprites:
1. Download 32x32 or 64x64 pixel art sprites
2. Name them according to the pattern: {character}_{action}.png
3. Update asset_manifest.json with the new paths

Recommended sources:
- OpenGameArt.org (free)
- itch.io (free/paid)
- Pixel Art Asset Packs

Current emoji fallbacks:
- Warrior: ⚔️
- Mage: 🔮
- Archer: 🏹
- Priest: ✝️
- Rogue: 🗡️
