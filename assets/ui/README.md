# UI Assets

## Current UI Elements (using emoji/CSS)

All UI elements are currently created with CSS gradients and emoji icons.

## To Add Custom UI

1. Button States: normal, hover, pressed
2. Frame/Panel backgrounds
3. HP/MP/XP bar textures
4. Icon spritesheet

## Recommended Sizes
- Buttons: 120x40 pixels
- Panels: 9-patch or scaleable
- Icons: 16x16 or 32x32
