# Tileset Assets

## Current Implementation

Map is currently rendered as a grid of emoji/clickable zones.

## To Add Tilesets

1. Download terrain tilesets (grass, dirt, water, stone, lava)
2. Create 16x16 or 32x32 tile grids
3. Update map rendering to use canvas tile rendering
